# 中文核心论文通用模板

本文件夹用于沉淀高质量中文核心期刊论文的通用写作规范，适合系统设计、仪器测试、嵌入式实现、数据采集、测控、电子信息、自动化等工程类论文。

建议使用顺序：

1. 先看 [01_中文核心论文通用指令.md](G:\最终版\07_中文核心论文通用模板\01_中文核心论文通用指令.md)，明确总原则。
2. 再按 [02_中文核心论文章节模板.md](G:\最终版\07_中文核心论文通用模板\02_中文核心论文章节模板.md) 搭建正文结构。
3. 写完后用 [03_中文核心论文投稿前检查清单.md](G:\最终版\07_中文核心论文通用模板\03_中文核心论文投稿前检查清单.md) 逐项核查。
4. 如需让大模型直接协助写稿或改稿，可直接使用 [04_大模型写作提示词模板.md](G:\最终版\07_中文核心论文通用模板\04_大模型写作提示词模板.md)。

适用范围：

- 高质量中文核心期刊
- 偏工程验证、系统实现、实验测试、方法设计类论文
- 需要兼顾学术表达、工程证据、图表规范、期刊格式的稿件

核心原则：

- 先有主线，再铺数据
- 先讲问题，再讲方法
- 先给证据，再下结论
- 多写参数、结果、边界，少写空泛评价
- 保持期刊论文语气，避免项目汇报语气
