# 一眼会用

这项 skill 的名字是：

`$cn-core-paper-2-0`

最简单的用法只有 3 种。

## 1. 直接让它改稿

示例：

`Use $cn-core-paper-2-0 把这篇稿子改成中文核心工程期刊风格，保持数据口径不变。`

适合：

1. 已经有初稿
2. 想整体变得更像正式期刊论文
3. 不想自己一点点说规则

## 2. 直接让它重构结构

示例：

`Use $cn-core-paper-2-0 按中文核心工程论文结构重写这篇文章，不要写成项目汇报。`

适合：

1. 现在稿子主线乱
2. 章节像流水账
3. 想先把框架搭正

## 3. 直接让它做投稿前总检

示例：

`Use $cn-core-paper-2-0 对这篇文章做投稿前终检，重点查摘要、结论、图表、公式、单位和术语一致性。`

适合：

1. 已接近定稿
2. 想最后排雷
3. 想避免低级格式问题

## 最稳的一句话

如果你懒得想怎么说，直接用这句：

`Use $cn-core-paper-2-0 按中文核心工程期刊要求处理这篇文章，先保主线和数据口径，再统一语言、结构、图表、公式和投稿风格。`
